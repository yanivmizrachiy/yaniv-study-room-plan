# RULES.md

## מקור אמת
הריפו הזה הוא מקור אמת יחיד לאתר חדר הלימוד של יניב.

## מצב נוכחי
- האתר מפורסם דרך GitHub Pages
- האתר החי:
  https://yanivmizrachiy.github.io/yaniv-study-room-plan/
- הריפו:
  https://github.com/yanivmizrachiy/yaniv-study-room-plan

## חוקים קבועים
- כל שינוי אמיתי חייב להיכנס ל-git עם commit ברור
- אין להכניס secrets לריפו
- כל טקסט בממשק בעברית RTL
- כל כפתור חדש חייב להיות אמיתי, ברור, ובטוח
- אין להצהיר על חיבור אוטומטי אם הוא לא נבדק בפועל
- כל שדרוג חייב לשמור על GitHub Pages פעיל
- האתר הציבורי לא ישמור token סודי בצד לקוח
- שינויים מבניים יתועדו גם ב-docs וגם ב-state

## מה נבנה בפועל
- אתר ציבורי חי
- חיבור לריפו אמיתי
- כפתורי GitHub בעברית
- תצוגת חדר אינטראקטיבית

## מה עדיין לא הושלם
- OAuth ל-GitHub מתוך האתר
- backend מאובטח
- אינטגרציה ל-server-core
- אינטגרציה ל-n8n
- טריגרים ישירים לשליטה במכשירים
