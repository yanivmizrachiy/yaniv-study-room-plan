# ארכיטקטורה

## שכבה 1 - אתר ציבורי
GitHub Pages מגיש את index.html.

## שכבה 2 - ניהול קוד
GitHub repository מנהל את קבצי האתר, היסטוריית שינויים, workflows ו-issues.

## שכבה 3 - אוטומציה
GitHub Actions מבצע בדיקות אתר, בדיקות תקינות, ואפשרויות הרחבה עתידיות.

## שכבה 4 - הרחבות עתידיות
- GitHub OAuth
- backend קטן ומאובטח
- אינטגרציה ל-server-core
- אינטגרציה ל-n8n
- הפעלת workflows מהאתר בצורה מבוקרת
